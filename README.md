# Required python modules:
- install pip

	$ easy_install pip

- install https://www.dlitz.net/software/python-pbkdf2/

	$ pip install pbkdf2

- install https://www.dlitz.net/software/pycrypto/

	$ pip install pycrypto

# Usage

- create database

	$ ./create_db

- edit your database

	$ ./edit_db

- change password

	$ ./password_db

- update html page

	$ ./update_html
